# Task Manager with Tag Filtering

A simple React-based Task Manager application that allows users to add tasks and filter them using tags. This project demonstrates state management, conditional rendering, and event handling in React.

---

## Features

- Add new tasks using task input field
- Select tags from dropdown while creating tasks
- Automatically reset input fields after adding a task
- Filter tasks by selecting tags
- Display all tasks when no tag is active
- Highlight active tag selection
- Responsive and user-friendly interface

---

## Functional Requirements

- Initially, the task list should be empty
- Task input field should be empty on load
- First tag option should be selected by default
- Clicking **Add Task** should:
  - Add task to the list
  - Reset Task input field
  - Reset Tag dropdown to default value
- Clicking a tag should:
  - Activate selected tag
  - Filter tasks based on selected tag
- If no tag is active:
  - All tasks should be displayed
